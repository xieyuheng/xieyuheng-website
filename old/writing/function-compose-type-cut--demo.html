  • 'the algebra of logic' first

    • an example of 'product'

      f : A B -> C D
      g : D F -> G H
      ----------------
      ? : F A B -> C G H

      F A B
            <- f
      F C D
            <- 1-2-swap
      C D F
            <- g
      C G H

          f        : A B -> C D
          g        : D F -> G H
      -----------------------------
      f 1-2-swap g : F A B -> C G H

    • another example of 'product'

      f : A B -> C D
      g : D F -> G H
      --------------------
      ? : A B D F -> C D G H

      A B D F
            <- g
      A B G H
            <- 2-2-swap
      G H A B
            <- f
      G H C D
            <- 2-2-swap
      C D G H

               f            : A B -> C D
               g            : D F -> G H
      ------------------------------------------
      g 2-2-swap f 2-2-swap : A B D F -> C D G H
